# Gen3 User Guide

Welcome to the **MMRF Virtual Lab User Guide**. This guide is designed for researchers, clinicians, and data scientists who use the MMRF Virtual Lab to explore, access, and analyze harmonized clinical and genomic data from MMRF research programs.  

It provides step-by-step instructions for navigating the platform, building cohorts, viewing data summaries, and downloading datasets for downstream analysis.

---

### Sections in This Guide

* [Cohort Builder](cohort-builder.md) - Learn how to filter and define participant cohorts based on clinical, molecular, and biospecimen attributes.
* [Repository](repository.md) - Instructions for accessing, filtering, and downloading data files available within the platform.
* [Mutation Frequency](mutation-frequency.md) - Explore gene- and variant-level summaries, mutation frequencies, and comparison tools.
* [Clinical Data Analysis](clinical-data-analysis.md) - Review integrated clinical data, visualize outcomes, and correlate findings with genomic features.
* [Cohort Comparison](cohort-comparison.md) - Compare two or more cohorts side-by-side to assess differences across key variables.

---

## Resources

* The [MMRF Virtual Lab Portal](https://mmrfvirtuallab.org) is the main access point for exploring, analyzing, and downloading data.
* The [Multiple Myeloma Research Foundation (MMRF)](https://themmrf.org) leads data-sharing initiatives that make this platform possible.
